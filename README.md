# A股每日选股网站（多因子 + 消息面）

## 1) 功能
- 每日推荐 A 股股票（默认 Top10）
- 多维度评分：
  - 技术趋势（MA20/MA60）
  - 动量（20日/60日收益）
  - 交易活跃度（量比、换手）
  - 风险约束（波动率惩罚）
  - 消息面情绪（新闻关键词）
- Web 页面展示每只股票的推荐理由
- 自动日更（工作日 18:05）

## 2) 启动
```bash
cd a-share-stock-site
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

访问：`http://127.0.0.1:8080`

## 3) 接口
- `GET /api/recommendations`：返回最新推荐 JSON
- `GET /api/run-now`：手动触发刷新

## 4) 关于“最有效”
市场不存在永远最有效的单一策略。这里采用的是**多因子组合策略**，原因是：
- 单因子容易阶段失效；组合因子更稳健
- 既看价格行为（技术/动量），也看交易行为（量能/换手）
- 叠加消息面，避免只看K线
- 有风险惩罚，避免盲目追高高波动

建议配合回测周期化优化参数（如 candidate_n、权重、止损条件）。

## 5) 注意
- 当前运行环境若无法访问行情源，会自动写入“演示结果”。
- 在你的服务器/云主机上正常联网后，会自动切换为真实数据日更。
- 仅供研究学习，不构成投资建议。
